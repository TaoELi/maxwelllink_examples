MaxwellLink Examples
----------------------

This Github repo stores input and post-processing files of publications using the [MaxwellLink](https://github.com/TaoELi/MaxwellLink) package.

1. [implementation_2025/](./implementation_2025/): Ji, X.; Bocanegra Vargas, A. F.; Meng, G.; Li, T. E. **[MaxwellLink: A Unified Framework for Self-Consistent Light-Matter Simulations.](https://doi.org/10.1021/acs.jctc.5c02028)** *J. Chem. Theory Comput.* **2026**, *22* (6), 2725-2738.

2. [vsc_transport_2026/](./vsc_transport_2026/): X. Ji, and T. E. Li. **MNonlinear Freezing of Vibrational Polariton Transport via Mesoscale Simulations.** *Submitted* 2026.
